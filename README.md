# To-Do List Web App

📝 A simple and responsive to-do list app using HTML, CSS, and JavaScript.

## 🔧 Features
- ✅ Add tasks
- 📝 Click to mark as completed
- ❌ Right-click to delete
- 📱 Responsive design

## 🛠️ Tech Used
- HTML5
- CSS3
- JavaScript (Vanilla DOM)
